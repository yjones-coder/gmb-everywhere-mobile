# Extension Icons

This directory should contain the following icon files for the Chrome extension:

- `icon16.png` - 16x16 pixel icon
- `icon32.png` - 32x32 pixel icon
- `icon48.png` - 48x48 pixel icon
- `icon128.png` - 128x128 pixel icon

## Icon Requirements

- Format: PNG
- Background: Transparent or solid
- Style: Simple, recognizable icon representing data export or Google Maps
- Colors: Should work well on both light and dark themes

## Placeholder Files

The .png files in this directory are currently empty placeholders.
Replace them with actual icon files before loading the extension in Chrome.

## Icon Generation

You can create these icons using:
- Online icon generators (favicon.io, realfavicongenerator.net)
- Design tools (Figma, Adobe Illustrator, Inkscape)
- Or use existing icon sets (Material Icons, etc.)

## Example Icon Concept

A simple icon could show:
- A spreadsheet/document with location pins
- A download arrow over a map marker
- A clipboard with map elements
